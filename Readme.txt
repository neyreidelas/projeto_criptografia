# 🔐 Projeto de Criptografia em Python

Este projeto foi desenvolvido como atividade prática para demonstrar o uso de **algoritmos de criptografia** utilizando a linguagem Python. O foco está em três áreas principais:

- **Criptografia com Chaves Simétricas** (ex: AES via Fernet)
- **Criptografia com Chaves Assimétricas** (ex: RSA)
- **Funções Hash** (ex: SHA-256, MD5)

---

## 🧠 Objetivo

Aplicar e compreender diferentes formas de criptografia e segurança de dados, explorando algoritmos amplamente utilizados em sistemas reais.

---

## 📁 Estrutura do Projeto

```bash
projeto_criptografia/
├── simetrica.py      # Criptografia com chave única usando Fernet (AES)
├── assimetrica.py    # Criptografia com chaves pública e privada (RSA)
└── hash.py           # Geração de hash com SHA-256 e MD5
